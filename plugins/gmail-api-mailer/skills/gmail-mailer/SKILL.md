# Gmail Mailer

## Purpose

Send emails through Gmail API with OAuth tokens stored locally.

## Expected Setup

- Gmail API enabled in Google Cloud
- OAuth desktop client secret saved locally
- Token generated after first consent flow

## Current Status

This plugin scaffold exists, but the actual Gmail API sender has not been implemented yet.
